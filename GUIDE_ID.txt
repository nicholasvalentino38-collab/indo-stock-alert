Panduan singkat:
1. Buat repository GitHub baru.
2. Upload semua file dari folder ini.
3. Buat bot Telegram lewat @BotFather.
4. Chat bot sekali, lalu ambil chat_id via:
   https://api.telegram.org/bot<TOKEN>/getUpdates
5. Di GitHub repo buka Settings > Secrets and variables > Actions.
6. Tambah secrets:
   - TELEGRAM_BOT_TOKEN
   - TELEGRAM_CHAT_ID
7. Tambah variables:
   - TICKERS
   - MIN_RVOL
   - MIN_PRICE_CHANGE_PCT
   - MIN_BAR_VALUE_IDR
   - LOOKBACK_BARS
   - BAR_INTERVAL
8. Commit file workflow ke default branch.
9. Buka tab Actions lalu jalankan manual "Run workflow" untuk tes.
10. Jika berhasil, workflow akan jalan otomatis sesuai jadwal.

Catatan:
- Scheduled workflows hanya berjalan di default branch.
- Workflow terjadwal bisa delay saat GitHub sedang ramai.
- Public repo yang tidak ada aktivitas selama 60 hari bisa membuat scheduled workflow nonaktif otomatis.
